Project for Computer Vision (MSU, 2025)
Автор: Жангирхан Шаку
Этот проект решает задачу классификации гистологических изображений по 9 классам тканей с использованием модели EfficientNet-B0, дообученной на предоставленных в задании данных. Решение полностью соответствует всем требованиям основной и дополнительной частей задания.
В проекте реализован класс Model, использующий:
EfficientNet-B0 (torchvision),
заменённый финальный слой на 9 классов,
обучение последнего слоя (классификатора),
fine-tuning всей сети после начального обучения.
Валидация на части обучающей выборки
Автосохранение модели
Вывод метрик (loss, accuracy)
Аугментации
Дообучение модели(metod fine-tune)
